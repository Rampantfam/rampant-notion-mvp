# Local Testing Guide

1. Run the dev server:
   npm run dev

2. Visit /signup → create an account
   - Confirm new row in Supabase → Auth > Users
   - Confirm new row in public.profiles

3. Fill out /onboarding/organization
   - Confirm new row in public.clients
   - Confirm profile record updates with client_id

4. Log out → /login to confirm everything works
